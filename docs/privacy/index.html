StudyLock Privacy Policy

Effective date: May 26, 2026

Contact: your-email@example.com

StudyLock is a local-first Chrome extension that helps students run focused study sessions, block distracting websites, save academic sources, capture highlights, create flashcards, manage study tabs, and review local study activity.

Data stored locally

StudyLock stores study data locally in the user's browser using chrome.storage.local. This may include:

  • Study session names, courses, tasks, goals, durations, and timestamps
  • User-defined blocked and allowed domains
  • Saved source URLs, page titles, domains, citation draft metadata, notes, and tags
  • Selected text saved by the user as highlights
  • Flashcard front and back text
  • Saved tab group titles, URLs, and page titles
  • Local study analytics such as blocked attempts, bypasses, focused time, saved sources, highlights, flashcards, and tab groups

No account required

StudyLock does not require an account.

No backend or cloud sync

StudyLock does not send study data to a remote server. StudyLock does not currently provide cloud sync.

No sale or transfer of user data

StudyLock does not sell, rent, or transfer user data to third parties.

No external analytics or tracking

StudyLock does not use external analytics services, advertising trackers, or third-party tracking scripts.

No AI API

StudyLock does not send user study data to AI APIs.

Chrome permissions

StudyLock uses Chrome extension permissions only to provide its core study features:

  • storage is used to save study sessions, settings, sources, highlights, flashcards, tab groups, and local analytics.
  • tabs is used to save and restore current-window study tab groups and close distracting tabs when the user explicitly chooses to do so.
  • activeTab and scripting are used after user action to capture the current tab title, URL, and selected text for study materials.
  • Host permissions are used so StudyLock can check pages against the user's active blocklist or allowlist during a study session.

User control

Users can reset local StudyLock data from the extension's settings page. Users can also export their local StudyLock data as JSON.

Uninstalling the extension removes extension-local data according to Chrome's extension storage behavior.

Changes to this policy

This privacy policy may be updated as StudyLock changes. Any major changes will be reflected on this page.
